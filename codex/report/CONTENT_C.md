## 작업 목표
- `c_coffee.jsx`의 기존 커피 컵 애니메이션을 더 읽기 쉬운 구조로 정리한다.
- `COFFEE` 텍스트가 머그잔처럼 보이도록 형태를 강화한다.
- 마우스를 누르고 있는 동안 위에서 커피가 따라지고, 컵 안의 커피가 천천히 물결치며 차오르도록 만든다.
- 입력이 멈춘 뒤 일정 시간이 지나면 커피가 서서히 줄어들도록 감쇠 로직을 다듬는다.
- 전체 무드를 기존보다 레트로한 포스터/카페 사인 느낌으로 보강한다.

## 구현 내용
### 1. 애니메이션 구조 재정리
- 기존의 분산된 컵/물결/낙하 로직을 상태 기반 구조로 다시 정리했다.
- `createState`, `getMetrics`, `updateState`로 상태 생성, 반응형 치수 계산, 프레임별 갱신 책임을 분리했다.
- 배경, 받침, 컵 실루엣, 커피 채움, 윤곽선, 김, 낙하 줄기를 각각 별도 함수로 나눠 이후 미세 조정이 쉬운 형태로 정리했다.

### 2. 머그잔처럼 보이는 COFFEE 텍스트
- `COFFEE` 텍스트 자체를 컵 몸통으로 사용하고, 하단 반원과 손잡이를 추가해 머그잔 실루엣을 강화했다.
- 컵 외곽선, 크림색 본체, 하이라이트 선을 더해 단순 검은 텍스트보다 사물 형태가 더 잘 읽히도록 변경했다.
- 컵 아래에는 받침 접시와 그림자를 추가해 정적인 포스터 같은 안정감을 만들었다.

### 3. 커피가 따라지고 차오르는 인터랙션
- 마우스를 누르고 있는 동안만 낙하 줄기의 길이와 압력이 커지도록 변경했다.
- 줄기가 컵 표면에 닿으면 즉시 꽉 차는 대신 `targetLevel`이 조금씩 증가하고, 실제 수위 `level`은 완만하게 따라가도록 분리했다.
- 이 구조 덕분에 “붓는 입력”과 “컵 안에서 차오르는 결과”가 서로 다른 속도로 움직여 더 자연스럽게 보인다.

### 4. 물결과 감쇠 표현
- 컵 내부 수면은 사인 기반 물결선으로 그리고, 줄기가 닿을 때마다 `ripple` 값을 올려 파도가 살아나도록 했다.
- 입력이 멈춘 직후 바로 줄지 않고, 일정 idle 시간이 지나야 서서히 커피 수위가 내려가도록 설정했다.
- 오래 비워두면 조금 더 빠르게 줄어들도록 감쇠 속도를 2단계로 분리했다.

### 5. 레트로 무드 강화
- 청록 계열 배경 그라디언트, 밝은 원형 광원, 얇은 가로 스트라이프를 추가해 복고풍 포스터 느낌을 만들었다.
- 컵과 받침은 크림/브라운 계열로, 커피는 진한 갈색 그라디언트로 구성해 전체 톤을 따뜻하게 맞췄다.
- 커피가 어느 정도 찬 뒤에는 은은한 김이 올라오게 해서 정적인 장면에서도 분위기가 유지되도록 했다.

### 6. 콘텐츠 메타데이터 보강
- `items['C']`의 `backgroundColor`를 새 비주얼 톤에 맞게 조정했다.
- 비어 있던 `text`, `toolTipText`를 현재 애니메이션 의도에 맞는 내용으로 채웠다.

### 7. 후속 피드백 반영
- 커피 채움 영역을 컵 내부 사각 마스크로 다시 제한해, 커피가 잔 바깥으로 삐져나와 보이는 문제를 줄였다.
- 위에서 내려오는 커피 줄기는 캔버스 상단에서 시작하도록 바꾸고, 컵 안쪽 수면까지 이어지는 내부 줄기를 추가해 “잔 안으로 들어가는” 흐름이 더 잘 보이도록 수정했다.
- 배경의 가로 스트라이프는 더 이상 움직이지 않도록 고정했다.
