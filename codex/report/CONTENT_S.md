# CONTENT S 개발 보고서

## 개요
- 파일: `src/animate-js/class/animate/s_stubborn.jsx`
- 주제: `Stubborn` 인터랙티브 타이포그래피
- 목표: "고집"의 감각을 점토처럼 버티고 되돌아오는 글자 움직임으로 표현

## 작업 목표
- 화면 중앙에 `Stubborn` 타이포그래피를 배치한다.
- 글자를 직접 드래그하면 특정 글자 하나만 떨어져 움직이는 것이 아니라, 단어 전체가 연결된 덩어리처럼 유기적으로 반응하도록 만든다.
- 포인터를 놓으면 글자가 스프링처럼 원형으로 복원되도록 만든다.
- 배경을 드래그하면 단어 위치가 이동하지 않고, 제자리에서 압축되거나 인장되는 반응을 만든다.
- 이 전역 변형도 포인터를 놓으면 자연스럽게 원상복귀되도록 만든다.

## 최종 구현 요약
1. 중앙 타이포그래피 레이아웃만 남기고 배경 연출은 제거했습니다.
- 기존에 사용하던 패널, 종이 결, 광원형 배경 렌더를 걷어내고 `Stubborn` 자체의 형태 변화에 집중하도록 정리했습니다.
- 캔버스는 투명하게 두고, 페이지의 기본 종이 톤 위에 타이포그래피만 떠오르도록 맞췄습니다.

2. 중앙 타이포그래피 레이아웃을 유지하면서 글자 측정 구조를 정리했습니다.
- `Stubborn` 각 글자의 폭, ascent, descent를 캔버스에서 직접 측정해서 글자별 홈 위치를 계산했습니다.
- 단어 전체가 항상 중앙에 오도록 총 폭을 기준으로 정렬했습니다.

3. 글자 드래그 시 전체가 유기적으로 따라오도록 수정했습니다.
- 포인터 다운 시 현재 변형 상태를 반영한 글자 히트박스를 계산해서 어떤 글자를 잡았는지 판별합니다.
- 잡힌 글자의 드래그 벡터를 기준으로 전체 글자에 거리 기반 share를 나눠 주어, 가까운 글자는 많이 반응하고 먼 글자도 약하게 따라오도록 만들었습니다.
- 입력값은 그대로 사용하지 않고 `resist()` 함수와 스프링 감쇠를 거쳐 적용해서, 전체가 한 덩어리로 버티다가 늦게 따라오는 감각을 유지했습니다.

4. 점토 같은 늘어짐을 시각적으로 유지했습니다.
- 변형된 글자는 한 번만 그리지 않고, 홈 위치와 현재 위치 사이에 여러 겹의 trail을 찍어 늘어나는 덩어리처럼 보이게 했습니다.
- 최종 글자는 회전된 축 기준으로 늘어나는 스케일과 약한 shear를 적용해, 기계적으로 찌그러지는 느낌보다 손으로 잡아당기는 듯한 감각에 가깝게 맞췄습니다.
- 메인 글자에는 점토 질감이 느껴지도록 상단 하이라이트와 하단 음영이 있는 세로 그라디언트를 사용했습니다.

5. 배경 드래그를 전방향 단어 변형으로 다시 정리했습니다.
- 글자가 아닌 배경을 잡으면 전역 `groupPull` 스프링 상태가 반응하도록 분리했습니다.
- 포인터 다운이 시작된 위치에서 단어 중심으로 향하는 방향 벡터를 먼저 고정하고, 마우스 이동이 그 방향이면 압축, 반대 방향이면 인장으로 해석하도록 바꿨습니다.
- 이 전역 값은 각 글자에 따로 분배하지 않고, `Stubborn` 전체 레이어에 한 번의 축 방향 스케일로 적용되도록 바꿨습니다.
- 단어 바깥쪽 어느 지점을 잡아도 그 방향 축으로 회전-스케일-역회전이 적용되기 때문에, 상하좌우뿐 아니라 대각선 포함 모든 방향에서 한 덩어리처럼 눌리고 늘어납니다.

6. 복원 모션을 유지했습니다.
- 포인터를 놓으면 글자 개별 pull과 전역 pull 모두 목표값이 `0`으로 돌아가도록 했습니다.
- 각 값은 바로 초기화하지 않고 velocity를 가진 스프링으로 감쇠시키기 때문에, 딱딱하게 리셋되지 않고 탄성 있게 복원됩니다.

## 상호작용 로직
- `letter drag`: 잡은 글자의 드래그 벡터가 전체 글자에 거리 기반으로 전이되어, 단어 전체가 연결된 점토처럼 유기적으로 반응합니다.
- `background drag`: 시작 지점에서 글자 중심을 향하는 움직임은 압축, 반대 방향은 인장으로 해석하고, 개별 글자가 아니라 단어 전체가 한 덩어리로 변형됩니다.
- `release`: 글자별 spring과 전역 spring이 동시에 원점으로 수렴하며 원상복귀합니다.

## 시각 구성
- 짙은 차콜 계열 배경 위에 오프화이트 타이포그래피를 올려, 단어의 덩어리감과 압축/인장 변화가 더 선명하게 읽히도록 정리했습니다.
- 배경은 어두운 그라디언트와 은은한 중심광만 남기고, 글자 그림자는 제거해 형태 변화에 시선이 집중되도록 했습니다.
- 서체는 장식적인 세리프 계열을 사용해 고집스럽고 강한 인상을 유지했습니다.
