# CONTENT A 개발 보고서

## 대상
- 파일: `src/animate-js/class/animate/a_algorithm.jsx`
- 주제: `ALGORITHM` 타이포그래피 기반 버블 정렬 애니메이션 리디자인

## 작업 목표
- `ALGORITHM`이라는 단어가 화면의 중심 타이포그래피로 보이도록 구성
- 각 알파벳 위에 서로 다른 높이의 막대를 배치
- 막대가 처음에는 무작위 순서로 놓여 있다가 시간이 지나며 `bubble sort` 방식으로 정렬되도록 구현
- 마우스 입력 시 다시 셔플되어 정렬 애니메이션이 재시작되도록 구성
- 전체 분위기를 레트로하면서도 교과서/학습 자료 같은 인상으로 다듬기

## 최종 구현 요약
1. 버블 정렬 상태 관리를 새로 구성했습니다.
- `values`, `compareIndex`, `sortedTail`, `swapping`, `swapProgress`를 분리해 버블 정렬의 각 단계를 명확하게 관리하도록 정리했습니다.
- 단순히 값만 바꾸는 것이 아니라, 서로 비교 중인 인접 막대가 실제로 위치를 교환하는 것처럼 보이도록 애니메이션을 넣었습니다.

2. `ALGORITHM` 글자 위에 막대가 정확히 대응되도록 배치했습니다.
- 텍스트 전체 폭을 측정한 뒤 각 글자의 중심 좌표를 계산해, 각 알파벳 위에 막대가 올라가도록 구성했습니다.
- 결과적으로 막대가 단순한 차트가 아니라, 타이포그래피와 결합된 시각 요소처럼 보이게 했습니다.

3. 마우스 인터랙션을 정리했습니다.
- 마우스를 누르는 순간 한 번만 셔플이 일어나도록 `prevIsDown` 기반으로 처리했습니다.
- 누르고 있는 동안 반복 셔플이 발생하지 않도록 해 동작을 안정화했습니다.

4. 정렬 완료 상태를 보정했습니다.
- 정렬이 끝난 뒤에도 활성 비교 테두리나 붉은 강조색이 남던 문제를 수정했습니다.
- 정렬 완료 시 전체 막대를 sorted 상태로 처리하고, active 하이라이트는 비활성화되도록 정리했습니다.

5. 비주얼 스타일을 레트로 교재 느낌으로 조정했습니다.
- 배경은 베이지 계열 그라디언트와 비네트 중심으로 단순화했습니다.
- 초기 시안에 있던 뒤쪽 스트라이프와 원형 광원은 제거해 더 담백한 화면으로 정리했습니다.
- 중앙의 사각형 강조 영역은 단순한 면 채우기 대신, 교과서 표지처럼 보이는 이중 프레임 구조로 변경했습니다.
- 프레임은 글자와 막대의 톤에 맞는 색으로 정리하고, 이후 피드백에 따라 더 크게 키우고 선 두께도 보강했습니다.

6. 타이포그래피를 수정했습니다.
- 사용 중이던 italic 스타일은 제거했습니다.
- 현재는 `VT323` 기반의 픽셀풍 타이포를 사용해, 알고리즘/학습 자료와 잘 어울리는 복고적 인상을 주도록 맞췄습니다.

7. 세부 비례를 추가 조정했습니다.
- 프레임 좌우 여백은 조금 줄여서 타이포와 더 밀착되도록 조정했습니다.
- 막대 최대 높이는 소폭 늘려서 위쪽 리듬감이 더 잘 보이도록 조정했습니다.

## 세부 로직
- 초기 진입 시 `1~9` 값으로 구성된 막대 높이 배열을 셔플합니다.
- 매 프레임 현재 비교 인덱스를 기준으로 인접 원소를 비교합니다.
- 왼쪽 값이 더 크면 두 막대의 X 위치를 보간해 스왑 애니메이션을 수행합니다.
- 한 패스가 끝나면 뒤쪽 정렬 완료 구간을 확정합니다.
- 정렬 완료 후에는 추가 스왑 없이 안정된 상태를 유지합니다.
- 사용자가 마우스를 누르면 전체 상태를 초기화하고 다시 셔플부터 시작합니다.

## 결과
- `ALGORITHM`이라는 제목성과 정렬 알고리즘이라는 개념 연결이 이전보다 훨씬 직접적으로 드러나게 되었습니다.
- 버블 정렬이 “실제로 움직이며 정렬되는 과정”으로 보이도록 만들어 상징성과 전달력이 좋아졌습니다.
- 과한 장식은 줄이고, 교과서나 오래된 컴퓨터 학습서 같은 분위기를 가진 정돈된 레트로 화면으로 정리했습니다.

## 검증
- `react-scripts build` 기준으로 빌드가 정상 완료되는 것을 확인했습니다.
- 이후 사용자 피드백에 따라 시각 요소를 여러 차례 조정했으며, 최종적으로는 정렬 완료 하이라이트 잔존 문제와 프레임/배경 스타일 문제를 함께 반영한 상태입니다.
