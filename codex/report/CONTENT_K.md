# K - Kidult 개발 보고서

## 1. 작업 개요

- 대상 파일: `src/animate-js/class/animate/k_kidult.jsx`
- 작업 일자: `2026-03-29`
- 테마: `LEGO`
- 목표: 레고 판을 위에서 내려다보는 시선으로, `2 x 2` 레고 블록만 사용해 `KIDULT` 글자를 표현하는 애니메이션으로 재구성

## 2. 기획 방향

초기 버전은 "완성된 레고 글자를 보여주는 장면"에 초점을 둔 자동 조립형 애니메이션이었다.
이번 수정에서는 사용자가 직접 레고 블록을 집어서 다른 위치에 끼우는 놀이성이 더 중요해졌다.

핵심 방향은 아래와 같다.

- 기존보다 더 큰 레고 판 사용
- 탑뷰(top view) 유지
- `KIDULT` 글자를 `2 x 2` 블록으로 구성하되, 각 블록을 직접 드래그 가능하게 변경
- 원래 글자 위치는 희미한 가이드로 남겨 조립 놀이의 기준점 제공
- 배경은 베이지 계열 대신 차가운 블루 톤으로 교체
- 기존 상단 스트라이프 애니메이션 제거

## 3. 구현 내용

### 3.1 레이아웃 생성

- `K, I, D, U, L, T` 각 문자를 5행 패턴으로 정의
- 패턴의 `1` 값을 실제 블록 배치 데이터로 변환
- 문자 간 1칸 간격을 두고 전체 `KIDULT` 레이아웃 생성

이 방식 덕분에 추후 다른 단어로 바꾸거나 일부 문자의 형태를 수정하기 쉽다.

### 3.2 캔버스 드로잉 구조 재작성

`k_kidult.jsx`를 거의 새로 작성했다.
렌더링은 아래 순서로 구성했다.

1. 블루 톤 배경 그라디언트와 비네트
2. 회전이 아주 조금 들어간 레고 베이스 플레이트
3. 베이스 플레이트 전체 스터드
4. 원래 `KIDULT` 글자 위치를 알려 주는 빈 슬롯 가이드
5. 드래그 가능한 각 브릭 본체, 상단 면, 4개의 스터드, 하이라이트

배경의 수평 스트라이프 애니메이션은 제거했다.
색상은 베이지 대신 차갑고 밝은 블루 계열로 조정했다.

### 3.3 상호작용 구조 변경

- 포인터 다운 시 가장 위에 보이는 브릭을 선택
- 선택한 브릭은 드래그 중 최상단으로 이동
- 포인터 위치를 회전된 레고 판 좌표계로 역변환하여 정확한 드래그 처리
- 브릭 중심이 가까운 셀을 계산해 스냅 후보 생성
- 이미 다른 브릭이 차지한 칸은 배치 불가
- 유효한 칸 위에서는 반투명 스냅 프리뷰 표시
- 놓을 수 없는 곳에 드롭하면 원래 위치 유지

### 3.4 레고 판 확대

사용자 요청에 맞춰 레고 판 비율을 키웠다.
가로 사용 폭과 세로 사용 폭을 모두 늘려,
블록을 옮길 수 있는 여유 공간이 이전보다 더 넓게 보이도록 조정했다.

### 3.5 텍스트 리소스 정리

- `descriptionK`를 드래그형 놀이 경험 중심 서술로 수정
- `toolTipK`를 직접 끼우는 상호작용 설명 중심으로 수정
- 메인 메뉴의 `K` 항목 날짜를 `2026-03-29`로 반영
